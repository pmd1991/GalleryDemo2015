About

This site demonstrates use of Bootstrap, Angular, CSS and Html. Images were collected from my friend's flickr account.

A gallery template seeded the project with a bootstrap header and grid of thumbnails. http://startbootstrap.com

A grunt responsive images resized the images to be responsive.

An Angular directive from Angular UI Bootstrap was used for the slide show and dropdown.

An Angular factory holds a table of image file information rather than connecting to a database backend so the site could be hosted on github without a backend.

An Angular breadcrumb service was downloaded from github: http://ianvonwalter.com

Google maps was incorporated from a Udacity homework assignment.
